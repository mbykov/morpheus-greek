Morpheus for Ancient Greek:

  • offline web application for Windows, MacOS, Linux. Works offline, synchronizes when online
  • instantly displays any amount of text
  • uses the wiktionary.org data to create tests, so has several hundred thousand tests
  • analyzes the parts of the compound word, so allows in-depth analysis - prefixes, affixes, several stems, etc
  • The reverse side of this power - if there is no obvious correct result, Morpheus can find a chain of constituents that are formally correct, but do not make sense as a whole
  • But the cure for this is that
  • user can easily create / install / remove / publish the special dictionaries for every text by his own hands
